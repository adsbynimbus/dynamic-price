# Module Dynamic Price

Dynamic Price is a set of utilities and helper methods for including Nimbus in a Google Ad
Manager mediated auction using the [Google Mobile Ads SDK](https://developers.google.com/ad-manager/mobile-ads-sdk/android).

[Ad Manager setup instructions](https://docs.adsbynimbus.com/docs/extended-documentation/dynamic-price/ad-manager-setup)
are available in the Dynamic Price section of the Nimbus Extended Documentation.
